# Let Me Sleep!

InnoGames GameJam 2016\
Topic: Time Cheating\
Game Engine: Unity3D

Moorhuhn-like shooter/clicker game in which you need to maximize your sleeping time by scaring away the cuckoos!\
Keep clicking the cuckoos to keep the alarm clock from ringing!
